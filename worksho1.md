# ใบงาน Workshop: จัด Layout เว็บไซต์ยุคใหม่ด้วย CSS Flexbox 📦

**เป้าหมายของใบงานนี้:**
1. รู้วิธีดาวน์โหลดโค้ดเริ่มต้น (Clone) จาก GitHub ของครู
2. เลิกใช้ `float` และเลิกสุ่มตัวเลข `margin` ในการจัดเรียงหน้าเว็บ
3. จัดของให้อยู่กึ่งกลางเป๊ะๆ ได้ใน 3 บรรทัด!

## 🚀 ขั้นตอนที่ 0: เตรียมไฟล์ทำงาน (Clone Git Repository)

ก่อนเริ่มเขียนโค้ด เราจะดึงไฟล์โครงสร้างพื้นฐานที่ครูเตรียมไว้ให้ลงมาที่เครื่องคอมพิวเตอร์ของเราก่อน

**วิธีทำ:**

1. เปิดโปรแกรม **Terminal** (หรือ Command Prompt / Git Bash)
2. พิมพ์คำสั่งเพื่อเข้าไปยังโฟลเดอร์ที่ต้องการเก็บงาน เช่น หน้า Desktop:
   ```bash
   cd Desktop
พิมพ์คำสั่ง git clone ตามด้วย URL ของครู (ระวังเรื่องการเว้นวรรค):

Bash
git clone [https://github.com/ช](https://github.com/ช)ื่อผู้ใช้ของครู/ชื่อrepoของครู.git
(หมายเหตุ: นำลิงก์ที่ครูให้ในห้องเรียนมาใส่แทนที่ลิงก์ด้านบน)

เมื่อโหลดเสร็จแล้ว ให้เข้าไปในโฟลเดอร์งานที่เพิ่งโหลดมา:

Bash
cd ชื่อrepoของครู
เปิดโฟลเดอร์นี้ในโปรแกรม VS Code เพื่อเริ่มเขียนโค้ด:

Bash
code .
📦 การทดลองที่ 1: จัดกล่องเรียงแนวนอน (บอกลา Float และ Margin มั่วๆ)
ปัญหาที่เจอบ่อย: เวลาเราอยากให้กล่อง 3 ใบเรียงติดกัน เรามักจะติดนิสัยใช้ float: left; พอระยะมันชิดไปก็กะตัวเลขใส่ margin-right: 15px;... พอดูในมือถือ พัง! กล่องตกขอบ!

วิธีแก้ด้วย Flexbox: เราจะสั่งให้ "กล่องแม่" (Container) จัดการเรียงและเว้นระยะลูกๆ ให้เองแบบอัตโนมัติ

1. เปิดไฟล์ index.html จะเห็นโค้ดโครงสร้างนี้ที่ครูเตรียมไว้ให้:

HTML
<div class="box-container">
    <div class="box">1</div>
    <div class="box">2</div>
    <div class="box">3</div>
</div>
2. เปิดไฟล์ style.css แล้วพิมพ์โค้ดนี้เพิ่มลงไปในคลาส .box-container:

CSS
.box-container {
    /* 1. เสกให้กล่องแม่เป็น Flexbox เพื่อเรียงของแนวนอน */
    display: flex; 
    
    /* 2. สั่งเว้นระยะห่างทุกกล่อง 20px เท่ากันเป๊ะ (ไม่ต้องสุ่ม margin แล้ว!) */
    gap: 20px; 
}
👀 สังเกตผลลัพธ์: กดเซฟแล้วดูที่หน้าเว็บ กล่องทั้ง 3 ใบจะมาเรียงต่อกันในแนวนอนทันที และมีช่องว่างตรงกลางเท่ากันเป๊ะๆ โดยที่เราไม่ต้องบวกเลขกะระยะเองเลย!

🎯 การทดลองที่ 2: จัดของให้อยู่ "กึ่งกลาง" หน้าจอแบบ 100% เป๊ะ!
ปัญหาที่เจอบ่อย: อยากให้ข้อความหรือกล่องอยู่ตรงกลางหน้าจอเป๊ะๆ บางคนใช้ margin-top: 150px; margin-left: 200px; สุ่มตัวเลขไปเรื่อยๆ พอเปลี่ยนขนาดจอคอม อ้าว... เบี้ยวอีกแล้ว!

วิธีแก้ด้วย Flexbox: ใช้คำสั่งคู่หูมหัศจรรย์ คือ justify-content (จัดกลางแนวนอน) และ align-items (จัดกลางแนวตั้ง)

1. เปิดไฟล์ index.html ลองดูในส่วนนี้:

HTML
<div class="center-zone">
    <h1>สวัสดี Flexbox!</h1>
</div>
2. เปิดไฟล์ style.css แล้วพิมพ์โค้ดนี้เพิ่มลงไป:

CSS
.center-zone {
    /* กำหนดความสูงของพื้นที่ให้เต็มจอ (Viewport Height) */
    height: 100vh; 
    background-color: #f1f2f6;

    /* เริ่มใช้งาน Flexbox */
    display: flex;
    
    /* จัดให้อยู่กึ่งกลางแนวนอน (ซ้าย-ขวา) */
    justify-content: center; 
    
    /* จัดให้อยู่กึ่งกลางแนวตั้ง (บน-ล่าง) */
    align-items: center; 
}
